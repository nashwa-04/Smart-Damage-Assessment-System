# TASK: Setup API Documentation using Scribe

أريد إضافة توثيق تلقائي للـ API في مشروع Laravel السابق.

1. زودني بأمر تثبيت حزمة `knuckleswtf/scribe`.
2. اشرح لي كيف أقوم بنشر ملف الإعدادات (`vendor:publish`).
3. أعطني مثالاً لكيفية كتابة الـ DocBlocks (التعليقات) فوق دالة `store` في `ReportController` لتظهر في التوثيق بشكل صحيح (مثال: توضيح الـ Body Parameters وأن الصورة من نوع File).
4. ما هو الأمر لتوليد التوثيق (`php artisan scribe:generate`)؟ وكيف أصل لصفحة التوثيق محلياً؟
